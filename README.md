DataSet - https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud?resource=download


Dataset - The dataset has 30 features, where there is amount column which is providing details about transaction value. Class denotes as target column.
The features marked as V1-V28 are the transformed data provided by the credit card company. The feature names are not provided due to data privacy reason.
Amount indicates the transaction value of each transaction. 
Time denotes the time taken for the transaction. The actual time taken for the transaction can be found by deducting from the previous transaction time.


In this project data was imbalanced, smote was used to balance the data


Logistic regression is used for this classificcation problem



Datapipeline used for scalling the data


Neural network used to built the model
